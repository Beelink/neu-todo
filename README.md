# NeuTodo

Simple cross-platform todo app, built on top of Typescript, Vue, Vuetify and Neutralinojs

## Build app

##### 1. Install packages

```console
yarn
yarn update
```

##### 2. Develop

```console
yarn dev
```

Visit `http://localhost:8090/` to preview.

##### 3. Build binary

Build for web:

```console
yarn build
```

Build for desktop:

```console
yarn build && yarn desktop
```
